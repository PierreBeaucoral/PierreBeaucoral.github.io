Conflict data

Upload a UCDP GED export, an HDX conflict CSV, or any event-level dataset with a country, date, and count/severity column. We'll aggregate to annual country-level counts and chart them.

Note. UCDP's public API now requires an access token, and HDX's CDN restricts CORS to its own origin — so a pure-static site can't fetch them live. Two options:
  • Download the CSV from UCDP or HDX and drop it below.
  • Self-host the FastAPI backend (see api/ in the repo) and add an auth-equipped UCDP proxy route. Then point config.js → proxyBase at it.
