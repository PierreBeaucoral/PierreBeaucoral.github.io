# Introduction à l’économétrie — Quarto (site + livre)

## Pré-requis
- [Quarto](https://quarto.org) installé
- (Optionnel) LaTeX pour le rendu PDF du livre

## Lancer en local
```bash
quarto preview
# puis
quarto render --to pdf
```

## Données
Déposez les fichiers `TD1.xls`, `TD2.xls`, … dans `data/`.
